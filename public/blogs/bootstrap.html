Utilizando Bootstrap 4 en nuestro sitio

29 de Marzo del 2018 por Diego

Versión de Bootstrap en el momento de escribir este blog: v4.1.0

Pasos para utilizar Bootstrap 4 en nuestro sitio web:

1. Visitar el sitio oficial de Bootstrap:

Acceder getbootstrap.com para obtener los links de la versión más actualizada de Bootstrap 4.

Hay varias maneras de utilizar incorporar Bootstrap en nuestro proyecto:

  • Descargando los archivos y apuntarlos localmente
  • Apuntar directamente a los archivos de forma online mediante CDN

En este ejemplo utilizaremos el CDN, ya que nuestro sitio se encontrará disponible solo en forma online.

2. Linkear estilos y scripts:

Copiar las etiquetas de CSS y JavaScript a nuestra página en la sección BootstrapCDN.

Nota: Bootstrap requiere JQuery y Popper.js para su correcto funcionamiento, por esa razón conviene agregar las 3 etiquetas <script>.

3. Puesta en marcha:

Listo, ya podemos utilizar código de ejemplo de Bootstrap para nuestro sitio web.

En el sitio oficial de Bootstrap visitar la sección Documentation para poder agregar ejemplos a nuestra página.
